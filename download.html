Download

The latest version of Project: Starfighter can be downloaded via any of the following official sources:

  • Releases on GitHub (an Atom feed is also available)
  • GameJolt
  • itch.io

If you'd like to see how far the game has come, or if you'd simply like to see how the original game looked (with its unlicensed art assets), the Wayback Machine still has a copy of version 1.1, the last version released by Parallel Realities, archived. Do keep in mind that this old code is filled with bugs, including memory access bugs, albeit memory access bugs that shouldn't cause problems outside the game. It can be found here:

  • https://web.archive.org/web/20051016190326/http://www.parallelrealities.co.uk/download.php?file=starfighter-1.1-1.tar.gz&type=zip
